# Notes

This lab focuses on header metadata rather than payload content. Students should:

- inspect response headers carefully
- compare headers between secure and vulnerable implementations
- understand that the same body content can be safe or unsafe depending on the declared type
- verify that text payloads include `charset=UTF-8`

The vulnerable lab is intentionally functional but incorrect. Mistakes are limited to ASVS control 4.1.1 and should not introduce unrelated security issues.
